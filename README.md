## üéØ Overview

**EduCloudX** is a modern, full-stack social learning platform designed for computer science and engineering students. It offers a collaborative environment where learners can:

- Share educational resources and collaborate with peers
- Access organized course materials
- Engage with content through an intuitive and responsive interface
- Use integrated AI tools for smarter and faster learning

## ‚öôÔ∏è Setup and Installation

Follow these steps to run **EduCloudX** locally:

### 1. Prerequisites
- Python 3.8+
- Git

### 2. Clone the Repository
```bash
git clone https://github.com/Alan21303/EduCloudX-AI-Learning-Platform.git
cd EduCloudX-AI-Learning-Platform
```

### 3. Set Up a Virtual Environment
**On Windows**
```bash
python -m venv venv
.\venv\Scripts\activate
```

**On macOS/Linux**
```bash
python3 -m venv venv
source venv/bin/activate
```

### 4. Install Dependencies
```bash
pip install -r requirements.txt
```

### 5. Configure Environment Variables
Create a `.env` file in the project root and add:
```plaintext
SECRET_KEY='your-django-secret-key'
DEBUG=True
HUGGING_FACE_API_KEY='your-hugging-face-api-key'
```

### 6. Apply Migrations
```bash
python manage.py migrate
```

### 7. Load Initial Data (Optional)
```bash
python manage.py loaddata core_data.json
```

### 8. Run the Development Server
```bash
python manage.py runserver
```

Your app will be live at:
üëâ **http://127.0.0.1:8000**
